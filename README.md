## Overview
This project fetches **real-time earthquake data** from the [USGS Earthquake API](https://earthquake.usgs.gov/earthquakes/feed/v1.0/summary/all_day.geojson) 
and displays them on an interactive map using **React + Leaflet**.

## Features
- Live earthquake data from USGS API
- Interactive world map (pan & zoom)
- Markers showing location, magnitude, and time
- Responsive layout with title

## Tech Stack
- React
- React-Leaflet
- Leaflet
- CodeSandbox (deployment)
